Description

A short skill based combat game with rogue-like elements! Learn how to fight, clear the floors, buy upgrades and prepare for a showdown against the final boss!
The game is actually beatable with beginner weapon and no upgrades, but it promises to be challenging ;)

Hope you enjoy and good luck!

P.s the game is far harder on touchscreen devices.

Desktop Controls

  • WASD or Arrow Keys to move
  • LMB to attack
  • E to interact
  • Esc to pause

Touchscreen Controls

  • Left joystick to move
  • Tap the attack button to attack
  • The button to interact appears when near interactable objects
