Overview

SaveableObject is a component giving a prefab Save&Load functionalities, you should only attach one to a saveable object's root.

Properties

Name Description
Registry Name A unique name that the AssetsManager will register this SaveableObject by.
If the same name is registered more than once, the newer version will replace the older one.
Saveable Behaviours All ISaveables on the prefab should be assigned here, as otherwise they won't save nor load.
Subparts All SaveableSubparts on the prefab.
Bounds List of topdown 2d shapes that define which areas it occupy, for terrain binding and procedural generation purposes.
Most of the time, you should ensure that all colliders are contained within these bounds.
